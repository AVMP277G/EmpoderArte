# Generador de Frases Motivadoras Personalizadas

##  Descripción del proyecto

Este proyecto es una aplicación interactiva desarrollada en **Python** que genera frases motivadoras personalizadas a partir del **nombre** y la **meta** ingresados por el usuario.  

La aplicación combina lógica de programación con un toque artístico y emocional, integrando una pequeña interfaz gráfica (GUI) creada con **Tkinter**, donde cada clic transforma el ambiente visual y la energía de la frase mostrada.

El resultado es una experiencia dinámica, inspiradora y única cada vez que se ejecuta.

No es solo un programa…  
Es un recordatorio de lo que eres capaz de lograr.

Crear una herramienta que:

- Anime al usuario a perseguir sus metas

## Tecnologías utilizadas

- **Python 3**
- **Tkinter** (para la interfaz gráfica)
- **Random** (para generar variaciones aleatorias)

---

## Funcionalidades principales

 Solicita al usuario:

- Su **nombre**
- Su **meta personal**

 Genera frases motivadoras personalizadas usando:

- Listas de frases
- Concatenación de texto personalizada

 Cambia automáticamente al hacer clic o presionar Enter:

- Fondo de la ventana (tonos cálidos y vivos)
- Color del texto
- Tipo de letra (fuente)
- Tamaño de letra
- Frase mostrada

 Funciona con:

- Click izquierdo del mouse
- Tecla **Enter**

Cada cambio es aleatorio, por lo que cada interacción es única.

---

## Cómo ejecutar el programa

1. Asegúrate de tener **Python 3** instalado.
2. Guarda el archivo como:  
   `front.py`
3. En la terminal, ejecuta:

```
python front.py
```

4. Ingresa tu nombre y tu meta cuando el programa lo pida.
5. Haz clic en la ventana o presiona Enter para ver aparecer tu frase motivadora.

¡Deja que la inspiración haga el resto!
